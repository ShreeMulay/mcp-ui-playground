MCP-UI Playground

This playground requires the Bun dev server to run. It bundles React and the MCP-UI client library on the fly.

1
Install dependencies Run this in the playground directory
# Navigate to playground
cd ~/ai_projects/development/mcp_ui/playground

# Install dependencies
bun install
2
Start the dev server This will build and serve the app on port 4567
bun run dev

# Or specify a custom port:
PORT=8080 bun serve.ts
3
Open in browser The server URL will be shown in the terminal

Default URL: http://localhost:4567
